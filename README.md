# TingyunSDK

Tingyun iOS SDK 模块化分发包  
支持按需集成崩溃、网络、卡顿、主模块等功能。

---

## 🚀 支持模块

| 模块 | 名称 | 说明 |
|------|------|------|
| tingyunApp | 主模块 | SDK 核心接口 |
| tingyunCPU | CPU模块 |
| tingyunGM | 国密加密模块 |
| tingyunOOM | OOM监控模块 |
| tingyunReplay | 视频回放模块 |
| tingyunLog | 日志模块） |

---

## 📦 安装方式

### 通过 Swift Package Manager

1. 在 Xcode 中：  
   `File > Add Packages...`  
   填入仓库地址：https://github.com/stevension/tingyunSDK.git
